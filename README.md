## Graph Editor Framework
简介：基于阿里巴巴低代码引擎的统一图框架（图编排，图分析等等）

---

### 框架依赖
LCE：[lowcode engine]('https://lowcode-engine.cn/index')  
  
G6: [G6 图可视化引擎]('https://g6.antv.vision/zh')  
  
X6: [X6 图编辑引擎]('https://x6.antv.vision/zh')  


### 开始调试
#### 本地调试
`npm install`

`npm run setup`

`npm run start:x6`
